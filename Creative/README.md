## How to Submit

1. Read `CREATIVE_BRIEF.md` in full.
2. Create a file in the appropriate subdirectory using the naming convention: `[type]-[short-description]-[YYYYMMDD].md`
3. Include the required header in the file (see brief for format).
4. Add a row to the Submission Index above with status 📝.

**Subdirectories:**
- `Characters/` — profiles, histories, voice sketches
- `Vignettes/` — short scenes (100–600 words)
- `Stories/` — longer narratives (600+ words)
- `Quotes/` — standalone voiced moments (1–5 sentences, attributed)

---

## Evaluation Process

The project lead (Andy) reviews each submission against three criteria:

1. **Canon eligibility** — Does this enrich the permanent world without contradicting established canon?
2. **Flavor copy eligibility** — Could this be a card quote, Chronicle fragment, Narrator paragraph, or sidebar?
3. **World consistency** — Does it sound like New Meridian, feel like this city's specific texture and tone?

Canon decisions are recorded in PM02 (Decision Log). Flavor copy is tagged with a target artifact location in the index above.

A contradiction with established canon is not automatic disqualification — it may identify a gap in the canon worth addressing.
